# dotfiles

It's just my nvim (neovim) configuration 😳, don't laugh pls 🥲

## Install

Just clone with to `~/.config/nvim` and then: `:PaqInstall`

If you want to use my coc-nvim extensions as well, go ahead and install these:

* `coc-snippets`
* `coc-markdownlint`
* `coc-html`
* `coc-emoji`
* `coc-yaml`
* `coc-tsserver`
* `coc-toml`
* `coc-sql`
* `coc-solargraph`
* `coc-pyright`
* `coc-phpls`
* `coc-json`
* `coc-clangd`

with: `:CocInstall extension`

So simple, no?
